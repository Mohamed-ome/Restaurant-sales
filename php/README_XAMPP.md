# دليل تشغيل مشروع "المنطقة" باستخدام XAMPP

لقد قمت بربط المشروع بالكامل بقاعدة البيانات وتجهيز ملفات PHP لتعمل بشكل ديناميكي. اتبع الخطوات التالية للتشغيل على جهازك:

## 1. إعداد قاعدة البيانات (phpMyAdmin)
1. افتح لوحة تحكم **XAMPP** وقم بتشغيل **Apache** و **MySQL**.
2. افتح المتصفح واذهب إلى `http://localhost/phpmyadmin`.
3. قم بإنشاء قاعدة بيانات جديدة باسم `al_mantiqa_db`.
4. اختر قاعدة البيانات الجديدة، ثم اضغط على التبويب **Import** (استيراد).
5. اختر ملف `database.sql` الموجود داخل مجلد `php` في المشروع.
6. اضغط على **Go** أو **Execute** لإنشاء الجداول والبيانات التجريبية.

## 2. نقل ملفات المشروع
1. انسخ محتويات مجلد `php` من هذا المشروع.
2. الصقها داخل مجلد `htdocs` في مسار تثبيت XAMPP (غالباً `C:\xampp\htdocs\al_mantiqa`).

## 3. تشغيل المشروع
1. اذهب إلى المتصفح واكتب العنوان التالي:
   `http://localhost/al_mantiqa/index.php`

## 4. بيانات الدخول التجريبية (PIN)
*   **المدير (Admin):** `1234`
*   **المشرف (Manager):** `2222`
*   **الموظف (Waiter):** `1111`

## ملاحظات تقنية:
*   تم إنشاء ملف `includes/db.php` ليقوم بالاتصال التلقائي بقاعدة البيانات باستخدام تقنية **PDO** (الأكثر أماناً).
*   تم تحديث جميع الواجهات (نقطة البيع، لوحة التحكم، المخزون، التقارير) لتسحب البيانات حقيقية من الجداول التي أنشأتها.
*   المشروع يدعم اللغة العربية بشكل كامل (UTF-8) مع نظام Bootstrap RTL.
